# email-audit

Simple email authentication auditor.

Checks:

- MX
- SPF
- DKIM
- DMARC
- MTA-STS
- TLS-RPT
- DNSSEC

## Installation

```bash
git clone https://github.com/aiadam36/email-audit.git
cd email-audit

pip install -r requirements.txt
```

## Usage

```bash
python3 audit.py
```
